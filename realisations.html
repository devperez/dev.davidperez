Mes réalisations

Lili BOwL

Figma | HTML | CSS | Bootstrap | JQuery | Ajax | Laravel | SQL

Julie voulait un site sur lequel elle pourrait changer elle-même les menus. J'ai créé une partie administration qui lui permet, sans mon intervention, de faire ses changements, comme elle le souhaite. Elle peut également récupérer les adresses mails des internautes qui s'inscrivent à sa newsletter.
Le site lui permet de communiquer sur toutes les autres activités qu'elle mène de front en plus du restaurant.
Les miniatures ci-dessous illustrent cette partie "cachée" du site.

Mon site professionnel

Figma | HTML | CSS | Bootstrap | JQuery | Lightbox

C'est le site sur lequel vous êtes. Pour mon site professionnel, j'aurais pu choisir Wordpress et modifier un thème mais j'ai préféré partir de zéro. Je n'ai pas utilisé de framework parce que le projet est simple. J'ai tout codé en HTLM et CSS, j'ai rajouté un peu de Bootstrap pour le carrousel des recommandations en page d'accueil et la librairie Lightbox sur la pages des réalisations qui permet d'agrandir les images lorsqu'on clique dessus. Comme je ne voulais pas devoir créer un back office pour gérer les messages envoyés par le formulaire, j'ai externalisé leur traitement en passant par Formspree ce qui permet de gagner du temps.

DarkImmortal

HTML | CSS | Bootstrap | JQuery | Ajax | Laravel | Filament | Livewire | SQL

Grand fan de musique, j'ai plus ou moins toujours tenu un blog pour partager mes découvertes. Il y a quelques années, quand la plateforme de mon blog a fermé, j'ai perdu mon blog et pour éviter que ce genre de problème ne se reproduise, je me suis dit que la meilleure façon était de coder le site. C'est un projet qui est en cours et que j'améliore constamment. Les dernières améliorations concernent l'intégration de Filament côté back office et l'intégration d'un appel à l'API de LastFM côté front pour afficher mes derniers scrobbles. La page d'accueil affiche aussi un widget qui permet de récupérer le flux RSS d'un autre site pour l'afficher dans le footer. C'est un widget de news qui affiche les nouvelles des dernières 48 heures. Mon seul problème est le manque de temps pour publier plus régulièrement des articles.

Picture Box

HTML | CSS | React

Cette application est un projet que j'avais en tête depuis quelques temps. Le but ici était double : d'une part rendre "hommage" d'une certaine façon aux figures littéraires qui m'ont influencées pendant mon adolescence et mes années de jeune adulte et d'autre part jouer un peu avec la librairie React pour la prendre en main. Chaque carte est interactive. Au clic sur une photo, une pop-up s'affiche avec un court texte de présentation. Il est possible d'aggrandir les images du carrousel d'un simple clic.
